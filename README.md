# Tourist Spot

## Description

A simple webpage about tourism in South Korea.

This page aims to promote the location in question as a digital business card on the web.

## Requirements

No prerequisites.

## Installation

Steps to set up the development environment:

1. Clone the repository:
    ```sh
    git clone <https://github.com/dev-jvcosta/tourist-spot/tree/main>
    cd <tourist-spot>
    ```

2. Install the dependencies:
    ```sh
    npm install http-server
    ```

## Usage

Instructions on how to run the project:

1. To start the development server:
    ```sh
    npm start http-server
    ```

## Project Structure

- tourist-spot (main folder)
    - assets
